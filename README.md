# dreams 0.1.0

Make differentiable `treams` come true.
Automatic Differentiation for multi scattering problems.

## Usage

Use in conjunction with [jax](https://github.com/google/jax) to differentiate through treams.
See the [cluster of spheres optimization example](dreams/docs/examples/cluster_spheres_field.py)

## Installation

```bash
pip install git+https://gitlab.kit.edu/nigar.asadova/dreams.git
```
